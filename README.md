## Внедрение зависимостей
Библиотека [Microsoft.Extensions.DependencyInjection](https://www.nuget.org/packages/microsoft.extensions.dependencyinjection)
Основной инструмент для внедрения зависимостей в .net.
[Документация](https://learn.microsoft.com/ru-ru/dotnet/core/extensions/dependency-injection)

## Unit тесты
Библиотека [xUnit](https://www.nuget.org/packages/xunit) - основная библиотека для написания unit тестов в нашем приложении.
[Документация](https://xunit.net/docs/getting-started/netfx/visual-studio) - для xUnit.
[FluentAssertions](https://www.nuget.org/packages/fluentassertions) - для удобного написания Assert unit тестов.
Аналогичная библиотека для unit тестов в .net - [NUnit](https://www.nuget.org/packages/nunit)
